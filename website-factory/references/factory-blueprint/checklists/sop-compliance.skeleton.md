# SOP Compliance Checklist (per-niche layer)

This file is GENERATED by Module 2D from the niche wireframe + the
niche template's actual components. The factory's sop-qa-agent loads
this per-niche layer plus the universal layer at
`website-factory/.claude/checklists/sop-compliance.universal.md` (when
present) to compute the merged pass rate.

Score format: each item is `[ ] id, description` with a `Fix:` line for
automated remediation guidance.

Gate: 95% (PASS / (PASS + FAIL), excluding N/A) on the merged checklist.

---

## NICHE-SPECIFIC SECTION CHECKS

Module 2D fills the section list below from the niche wireframe. Each
section corresponds to a component generated in Phase 6c. The slot
syntax `{{section_name}}.{{check_id}}` becomes the actual check ID at
generation time.

For each section in the niche wireframe, this skeleton expands to:

```
## {{SECTION_LABEL}}

- [ ] `{{section_slug}}.composition_matches_wireframe`,
  {{section_slug}} renders the layout defined in
  `research/02-niche-research/{niche-slug}/09-wireframe.md` section
  `{{wireframe_anchor}}`.
  Fix: Inspect `templates/{niche-slug}/src/components/
  {{component_file}}.jsx` against the wireframe spec; the JSX
  structure should mirror the wireframe's annotated layout.

- [ ] `{{section_slug}}.brand_dna_paths_canonical`, the component
  reads only canonical brand-dna paths (no off-shape access).
  Fix: Run `npm run validate` and review reported paths.

- [ ] `{{section_slug}}.copy_from_playbook`, locked copy strings
  resolve from `templates/{niche-slug}/niche-playbook/copy-locks.json`
  (where applicable per the wireframe).
  Fix: Verify the component reads via canonical `brand-dna.copy.*`
  paths Stage 10.1 fills from the playbook.

- [ ] `{{section_slug}}.no_em_dashes`, zero em-dashes in rendered
  output.
  Fix: Replace em-dashes with colons, periods, or commas.
```

## GLOBAL CHECKS

These checks apply to every niche regardless of wireframe.

- [ ] `global.em_dash_audit`, `grep -rn` for em-dash + `&mdash;` +
  `&#8212;` across `src/` returns 0.
  Fix: Replace with safe punctuation.

- [ ] `global.no_placeholders`, no placeholder filenames anywhere in
  `public/`.
  Fix: Stage 4 / Stage 9 / manual drop required.

- [ ] `global.cta_anchors_form`, every primary CTA either anchors to
  the on-page form (`#lead-form`) or routes to `/contact`. Niche-
  specific component should follow the niche wireframe's CTA pattern.
  Fix: Inspect the niche wireframe's CTA pattern + adjust the
  component.

- [ ] `global.real_logo_present`, client logo is a real SVG/PNG.
  Fix: Stage 4 asset-scraper / manual drop.

- [ ] `global.font_loaded`, heading + body fonts load via Google
  Fonts per `brand-dna.typography.headingFontUrl` /
  `brand-dna.typography.bodyFontUrl`.
  Fix: Verify `index.html` <link rel="preconnect"> + the
  `@import url(...)` in `src/index.css` after `inject-theme.mjs`
  runs.

- [ ] `global.lighthouse_mobile_score`, Lighthouse mobile performance
  >= 80 (informational; flags but does not halt by itself).
  Fix: `tools/optimise-image.py`, lazy-load, code-split.
