# Design Fidelity Checklist (per-niche layer)

GENERATED by Module 2D from the niche wireframe + Phase 4 winner
screenshots. Carries the region SSIM thresholds, weighted means, and
composition checks specific to THIS niche's template.

The factory's design-fidelity-qa-agent (Stage 10.4a) loads this
per-niche layer plus the universal layer at
`website-factory/.claude/checklists/design-fidelity.universal.md`
(when present).

## NICHE-SPECIFIC REGION THRESHOLDS

Module 2D derives the region table below from the niche wireframe.
Sections that the wireframe places "above the fold" or near the
primary CTA get higher SSIM thresholds + higher weight. Section
weights are normalised so the weighted mean target stays 0.90.

For each section in the wireframe, this skeleton expands to:

```
| {{SECTION_LABEL}} | {{THRESHOLD}} | {{WEIGHT}} | {{REASON}} |
```

Example region table (filled per niche):

| Region | Threshold | Weight | Why |
|--------|-----------|--------|-----|
| {{HERO_REGION_LABEL}} | {{HERO_THRESHOLD}} | {{HERO_WEIGHT}} | First impression |
| {{TRUST_REGION_LABEL}} | {{TRUST_THRESHOLD}} | {{TRUST_WEIGHT}} | Trust establishment |
| ... | ... | ... | ... |

Aggregate target: 0.90 weighted mean across all regions, with no
region below its threshold.

## NICHE-SPECIFIC COMPOSITION CHECKS

Module 2D extracts the binary checks below from the niche wireframe.
Each check confirms the rendered build matches a specific layout
decision documented in the wireframe.

- [ ] `composition.section_order`, the rendered page matches the
  section order in
  `research/02-niche-research/{niche-slug}/09-wireframe.md`
  Fix: Inspect `src/pages/HomePage.jsx` against the wireframe.

- [ ] `composition.above_fold`, the wireframe's above-fold section
  ends above the 900px desktop fold.
  Fix: Adjust the above-fold section's min-height.

For each section in the wireframe with composition requirements:

```
- [ ] `composition.{{section_slug}}.{{check_id}}`, {{description}}
  Fix: {{fix_pointer}}
```

## PASS CRITERIA

1. Every per-niche region threshold passes.
2. Aggregate weighted mean meets 0.90.
3. Every composition + regression + image + mobile + performance
   gate passes.

Halts with `GENERATION-FAILED.md` if Module 2D could not fill the
region weights or composition checks. There is no shared region
rubric to fall back to.
